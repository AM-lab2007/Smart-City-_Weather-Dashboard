# Smart-City-_Weather-Dashboard
## 📌 Project Description

The Smart City Weather Dashboard is an interactive web application that provides users with real-time weather insights and a detailed 5-day forecast for any city around the world. Designed with a user-centric approach, the application not only displays essential weather parameters such as temperature, humidity, and conditions, but also enhances usability by offering smart, context-aware suggestions like what to wear or carry based on the current weather.

Users can easily search for a city or allow the application to automatically detect their location using the browser’s Geolocation API for a seamless experience. The dashboard further improves data exploration by enabling filtering of forecast data (such as rainy, hot, or cold days) and sorting based on parameters like temperature and humidity.

With a dynamic and visually responsive interface that adapts its background based on weather conditions, the application aims to deliver both functionality and an engaging user experience in a simple and efficient manner.
